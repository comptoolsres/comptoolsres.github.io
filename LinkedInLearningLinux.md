## Accessing LinkedIn Learning

From the [e-learning page](https://elearning.ufl.edu/), click the LinkedIn Learning link and login with your GatorLink.

Then either [click on this link](https://www.linkedin.com/learning-login/share?account=41282748&forceAccount=false&redirect=https%3A%2F%2Fwww.linkedin.com%2Flearning%2Flearning-linux-command-line-26594217%3Ftrk%3Dshare_ent_url%26shareId%3DBxz0Cq8cS1mJRTCxOd7Sqg%253D%253D), or search for the course Learning the Linux Command Line. This LinkedIn Learning tutorial will go over some basic Linux command line usage.
